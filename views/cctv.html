Video Surveillance

View and record video from up to four cameras

I wanted to create a video surveillance system where I could record video from multiple cheap USB and IP cameras simultaneously with scheduled and motion activated recordings.

Implementation

This was my final year Computer Science Degree project, built with .NET Winforms. Images are retrieved and event based image updates allow the captured image to be processed for on screen information, private area definition and motion detection before being displayed on screen. Scheduled and motion activated recordings can be configured and a search option displays all recordings on a selected date which can be viewed using the in built video player.

Any IP camera that allows for jpeg image capture can be used and by running a simple service on the PC with a connected USB camera we can treat it as an IP camera.

Motion detection is implemented by comparing the previous and current image captures using pixel XOR comparison. If the number of pixels different are above a configured threshold, motion has triggered.

Video clips can be saved using AVIWriter to create an avi file from multiple images saved per second and the resultant files played back using VLC Player library.
